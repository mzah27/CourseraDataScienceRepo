The FinalResults data set contains the data resulted after cleaning and merging different data files according to the instructions. The dataset includes:1) SubjectId:  includes id of participants2) ActivityId: includes id of the activity that is monitored (e.g. Walking, Laying, Sitting)3) ActivityName: contains the description of the activity based on provided Ids. (e.g. LAYING = 6 )The rest of the columns contain the Average of values provided for mean and standard-deviation variables in the source data. For instance the column tBodyAcc-meanX in FinalResults dataset contains the average of the corresponding column provided in both test and train datasets grouped by subject Id and Activity Id.
