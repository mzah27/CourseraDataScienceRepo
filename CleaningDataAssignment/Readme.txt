This submission includes four main files:run_analysis.R:  contains all the scripts ran step by step to generate the FinalResults data set.FinalResults.txt:  is a dataset that includes all the data cleaned and extracted from source materials in .txt format.FinalResults.xlsx: is a dataset that includes all the data cleaned and extracted from source materials in .xlsx format.CodeBook.txt:  includes all the information about the structure of FinalResults dataset.
